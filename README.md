### 🔹 Semantica — OSS Contributor

Open-source **semantic layer and knowledge engineering framework** for building explainable, auditable, and trustworthy AI systems. Contributes to context graph infrastructure, semantic retrieval pipelines, and knowledge engineering tooling used across AI stacks.

**⚡ Highlights**
- Context graphs with causal links and decision intelligence
- GraphRAG pipelines with full W3C PROV-O provenance
- Reasoning engines: forward chaining, Rete, SPARQL, deductive
- MCP server with 12 tools for Claude, Cursor, VS Code integration
- Semantic deduplication, entity resolution, and conflict detection
- Knowledge Explorer UI — React 19 + Sigma.js live graph canvas

**Tech Stack:**  
`Python` `Knowledge Graphs` `GraphRAG` `Neo4j` `SPARQL` `FastAPI` `MCP`
